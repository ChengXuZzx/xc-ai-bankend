# 天气查询技能

## 功能说明

本技能用于获取指定城市的实时天气信息。

## 使用方法

1. 确定用户要查询的城市名称
2. 调用天气查询API获取数据
3. 将结果整理成自然语言返回给用户

## 参数说明

- **city** (必填): 城市名称，如"成都"、"北京"、"上海"、"广州"等

## 返回信息

- 城市名称
- 当前温度（摄氏度）
- 天气状况（晴、多云、阴、雨等）
- 湿度（百分比）
- 风向和风力
- 数据更新时间

## 示例

用户问："成都今天天气怎么样？"

调用参数：
```json
{
  "city": "成都"
}
```

返回示例：
> 成都当前天气：多云，温度22℃，湿度65%，东南风2级，数据更新时间：2024-01-15 14:30:00

## API信息

- 接口地址：`https://restapi.amap.com/v3/weather/weatherInfo`
- 请求方式：GET
- 需要API Key：从环境变量 `WEATHER_API_KEY` 获取
